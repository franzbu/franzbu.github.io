HomeAssistantHeating

Python
ESP32
Modbus
Home Assistant

A sophisticated automation bridge designed to interface with Froeling heating systems. This project leverages the ESP32 platform to communicate over industrial Modbus protocols, exposing critical telemetry and control parameters directly to Home Assistant.

Core Capabilities:

  • Real-time monitoring of boiler temperatures, oxygen levels, and buffer state.
  • Bidirectional communication allowing for remote set-point adjustments.
  • Native Home Assistant integration via custom Python components and MQTT.
  • Optimized for low-latency feedback loops in local heating grids.

EnhancedSpaces.spoon

Lua
macOS
Hammerspoon

An advanced window and space management plugin for the Hammerspoon automation engine on macOS. This tool addresses the limitations of Apple's native Mission Control by providing a programmatic interface for complex window arrangements.

Key Features:

  • Dynamic Space Assignment: Automatically move specific applications to designated spaces upon launch.
  • Context-Aware Layouts: Switch entire window sets based on the active workflow (Development, Media, Admin).
  • Space-Manager Integration: Seamlessly hooks into macOS internal APIs to provide faster transitions and window-state persistence.
